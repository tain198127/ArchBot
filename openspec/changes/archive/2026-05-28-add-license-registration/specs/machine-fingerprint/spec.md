## ADDED Requirements

### Requirement: 系统启动时自动采集硬件指纹

系统启动时，Rust 后端 SHALL 自动采集本机硬件信息并生成稳定的 16 进制注册码，同一台机器每次启动生成的注册码完全一致。

#### Scenario: 正常采集生成注册码

- **WHEN** 系统启动且本机有可用网卡
- **THEN** 系统采集首个非 loopback 网卡的 MAC 地址和主机名
- **AND** 对采集信息做 SHA-256 哈希并取前 16 个 hex 字符作为注册码

#### Scenario: 同一机器多次启动生成相同注册码

- **WHEN** 在同一台机器上多次启动系统
- **THEN** 每次生成的注册码完全一致

#### Scenario: 不同机器生成不同注册码

- **WHEN** 在硬件配置不同的两台机器上分别启动系统
- **THEN** 两台机器生成的注册码不同
