## ADDED Requirements

### Requirement: 注册对话框展示机器注册码

用户通过菜单打开注册对话框时，系统 SHALL 展示当前机器的注册码（只读）和验证码输入框。

#### Scenario: 未注册状态打开注册对话框

- **WHEN** 用户点击"文件 → 注册"菜单项（当前为未注册状态）
- **THEN** 注册对话框显示本机 16 位 hex 注册码（只读，可复制）
- **AND** 显示验证码输入框和一个"注册"按钮
- **AND** 提示用户将注册码发送到指定邮箱

#### Scenario: 已注册状态打开注册对话框

- **WHEN** 用户点击"文件 → 注册"菜单项（当前已注册）
- **THEN** 对话框显示"已注册"状态
- **AND** 显示注册码和注册时间

### Requirement: 验证码输入与提交

用户 SHALL 能够在验证码输入框中输入 16 进制验证码并提交校验。

#### Scenario: 输入有效验证码

- **WHEN** 用户输入正确的验证码并点击"注册"按钮
- **THEN** 系统校验通过并显示"注册成功"
- **AND** 系统写入 license 文件并切换为已注册状态
- **AND** 所有受限功能变为可用

#### Scenario: 输入无效验证码

- **WHEN** 用户输入错误的验证码并点击"注册"按钮
- **THEN** 系统显示错误提示"验证码无效"
- **AND** 注册状态不变
