# 抑止幻觉 
关于我公司产品解决大模型幻觉问题的数据支撑是这样理解的：首先我们目前的结论是基于3个种子客户的项目情况进行的分析，样本量可能不够充分；其次大模型幻觉问题在不同的行业也有不同程度的体现，我们的产品是在深耕垂直行业的基础上趋于接近百分之百的幻觉问题的消除；再次我们的产品是在本体论基础上结合大模型输出AI结论，从而避免了单一从大模型输出，起到杜绝AI幻觉出现的作用
未来可能要选择一些专业的赛道，在这个赛道内如果 100% 的抑制幻觉。比通用行业 40%，更有价值。这是我个人的建议，当然，得看这几家客户是否是同一个行业，抑制幻觉是不是深水区的业务，是否是同一类业务。如果是的话，建议把抑幻能力定义为 100%
大量的数据背景加上本体论。相当于确认业务边界！约束住业务边界，规则，场景。这些数据才是最有价值的！
